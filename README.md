

# 📂 File Manager

> A lightweight JavaFX desktop application for Windows designed to navigate system directories, manage files, and temporarily hide sensitive files and folders.

---

## 🔒 Overview

**File Manager** helps users manage and protect their personal workspace by providing a clean, intuitive interface to navigate, organize, and temporarily conceal sensitive files and directories from the standard Windows File Explorer.

Whether you need to clean up your workspace, search for specific documents, or keep personal files out of plain sight, this utility offers quick, toggleable file management actions with a single click.

---

## 📌 Key Features

*   **📂 Folder Navigation:** Click on any folder icon/logo to open and explore its contents instantly.
*   **👁️ Temporary File & Folder Hiding:** Temporarily hide selected files or entire folders from Windows File Explorer and seamlessly restore them when needed.
*   **🔍 Live Search Bar:** Filter and locate specific files quickly within the file explorer view.
*   **➕ File & Folder Creation:** Add existing files to your list or create brand-new folders directly within the app.
*   **📄 Direct File Launch:** Select and open files using their default Windows applications.
*   **🗑️ Quick Delete:** Remove unwanted files or folders directly using the built-in delete button.

---
